Whe we resize the browser The insects names row should be breaks first.

Because the Bootstrap grid system uses a series of containers, rows, and columns. This grid system supports a max value of 12 columns.
Anything after the 12th column will be shifted to a new line.
1. Intailly the browser size is more than 1000 px, But when we resize the browser the pixel size will be reduced.
2. And we used the column with sizes of Small and Medium
3. As we go with the Bootstrap standards col-md px is 768. Anything below 768 px the row will be converts as column. That's why when decreasing the browser size first it cross the 768 px.
That is the reason Insert Names row breaks first
